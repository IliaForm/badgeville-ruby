Overview

Handles the fact that a JSON formatted GET response does not meet the ActiveResource standard, and is instead preceded by the root key :data.

def decode(json)  
  ActiveResource::Formats.remove_root(ActiveSupport::JSON.decode(json))["data"] || ActiveResource::Formats.remove_root(ActiveSupport::JSON.decode(json))
end
